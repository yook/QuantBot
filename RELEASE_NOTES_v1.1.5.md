# v1.1.5 (Русский)

## Основные изменения
- Добавлена защитная фильтрация по `target_query = 1` во всех worker-скриптах (типизация, категоризация, кластеризация) — исключены нецелевые запросы.
- Унифицирован курсор при наведении на перемещаемые заголовки столбцов: теперь такой же, как у фиксированных.
- Уточнено описание параметров кластеризации (`eps` / `threshold`).
- Улучшены сообщения об ошибках при ограничениях OpenAI (429 / недоступен ключ) — единый понятный текст.
- Мелкие UI-правки и стабильность worker-процессов.

## Детали
- `worker/trainAndClassify.cjs`, `worker/assignCategorization.cjs`, `worker/clusterСomponents.cjs`: фильтрация входных keywords по флагу `target_query`.
- `src/components/DataTableFixed.vue`: правка курсора при hover на draggable-заголовках.
- Улучшено логирование и проверка категорий перед запуском процессов.

## Обновление
1. Обновите до версии `v1.1.5`.
2. Убедитесь, что заданы целевые запросы (`target_query = true`).
3. Для кластеризации подберите параметры: компоненты (threshold ≈ 0.8) или DBSCAN (eps ≈ 0.2, minPts 2–5).

## Примечание
Если OpenAI ключ отсутствует или истек лимит — процессы классификации/категоризации/кластеризации завершаются с понятным сообщением.
